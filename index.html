Introduction

The ZoomSwiper component is a responsive image carousel built on top of SwiperJS. It features a thumbnail navigation strip and a high-performance "magnifying glass" effect that renders a zoomed preview in an external container based on the cursor position.

Live Interactive Demo

Hover over the main image to see the zoom effect on the right panel.

Installation

Include Swiper CSS and JS in your project before adding the custom logic.

<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/swiper@10/swiper-bundle.min.css" />

<!-- JS -->
<script src="https://cdn.jsdelivr.net/npm/swiper@10/swiper-bundle.min.js"></script>

HTML Structure

The structure consists of a wrapper containing two Swiper instances (Main & Thumb) and a separate Target Div for the zoom effect.

<div class="carousel-container">
    <!-- Main Swiper -->
    <div class="swiper main-swiper">
        <div class="swiper-wrapper">
            <div class="swiper-slide"><img src="image1.jpg"></div>
            <div class="swiper-slide"><img src="image2.jpg"></div>
        </div>
        <div class="swiper-button-next"></div>
        <div class="swiper-button-prev"></div>
    </div>

    <!-- Thumbnail Swiper -->
    <div class="swiper swiper-thumbnail">
        <div class="swiper-wrapper">
            <div class="swiper-slide"><img src="image1.jpg"></div>
            <div class="swiper-slide"><img src="image2.jpg"></div>
        </div>
    </div>
</div>

<!-- Zoom Target -->
<div id="target-div"></div>

JavaScript Logic

This script handles the synchronization between the two sliders and calculates the background position for the zoom effect.

// 1. Initialize Thumbnails
const thumbSwiper = new Swiper('.swiper-thumbnail', {
    slidesPerView: 4,
    spaceBetween: 10,
    watchSlidesProgress: true,
});

// 2. Initialize Main Slider
const mainSwiper = new Swiper('.main-swiper', {
    loop: true,
    autoplay: { delay: 2000, disableOnInteraction: false },
    navigation: {
        nextEl: '.swiper-button-next',
        prevEl: '.swiper-button-prev',
    },
    thumbs: { swiper: thumbSwiper },
});

// 3. Stop Autoplay on Hover
const container = document.querySelector('.main-swiper');
container.addEventListener('mouseenter', () => mainSwiper.autoplay.stop());
container.addEventListener('mouseleave', () => mainSwiper.autoplay.start());

// 4. Zoom Logic
const targetDiv = document.getElementById('target-div');
const slides = document.querySelectorAll('.main-swiper .swiper-slide');

slides.forEach(slide => {
    // Show Zoom Box & Set Image
    slide.addEventListener('mouseenter', () => {
        targetDiv.style.display = 'block';
        const img = slide.querySelector('img').src;
        targetDiv.style.backgroundImage = `url(${img})`;
    });

    // Hide Zoom Box
    slide.addEventListener('mouseleave', () => {
        targetDiv.style.display = 'none';
    });

    // Calculate Background Position
    slide.addEventListener('mousemove', (e) => {
        const rect = slide.getBoundingClientRect();
        // Calculate cursor position relative to image
        const x = e.clientX - rect.left;
        const y = e.clientY - rect.top;
        
        // Convert to Percentage
        const posX = (x / rect.width) * 100;
        const posY = (y / rect.height) * 100;
        
        // Move background image inside target div
        targetDiv.style.backgroundPosition = `${posX}% ${posY}%`;
    });
});
