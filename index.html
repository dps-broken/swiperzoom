Introduction

This documentation explains how to implement a Swiper Carousel with Thumbnail Navigation and a Hover Zoom Effect. Perfect for E-commerce product galleries.

Live Preview

Hover over the main image to see the zoomed details on the right.

Installation

Include Swiper CSS and JS in your project before adding the custom logic.

<!-- CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/swiper@10/swiper-bundle.min.css" />

<!-- JS -->
<script src="https://cdn.jsdelivr.net/npm/swiper@10/swiper-bundle.min.js"></script>

1. HTML Structure

Create a container for the carousel and a separate div for the zoom target.

<!-- Main Container -->
<div class="carousel-container">
    
    <!-- Main Swiper -->
    <div class="swiper main-swiper">
        <div class="swiper-wrapper">
            <div class="swiper-slide"><img src="image1.jpg"></div>
            <div class="swiper-slide"><img src="image2.jpg"></div>
        </div>
        <div class="swiper-button-next"></div>
        <div class="swiper-button-prev"></div>
    </div>

    <!-- Thumbnails -->
    <div class="swiper swiper-thumbnail">
        <div class="swiper-wrapper">
            <div class="swiper-slide"><img src="image1.jpg"></div>
            <div class="swiper-slide"><img src="image2.jpg"></div>
        </div>
    </div>
</div>

<!-- Zoom Target -->
<div id="target-div"></div>

2. CSS Styles

Use the following CSS to style the layout and handle the hidden zoom div.

/* Container Setup */
.carousel-container {
    width: 700px;
    margin-bottom: 20px;
    position: relative;
    max-width: 100%; /* Makes it responsive */
}

.swiper {
    width: 100%;
    height: 400px;
}

.swiper-slide img {
    width: 100%;
    height: 100%;
    object-fit: cover;
}

/* Zoom Target Box */
#target-div {
    height: 400px;
    width: 400px;
    background-color: #3e3e3e;
    display: none; /* Hidden by default */
    background-size: cover;
    position: relative;
}

/* Navigation Arrows */
.swiper-button-next,
.swiper-button-prev {
    color: red;
}

/* Thumbnails */
.swiper-thumbnail {
    width: 700px;
    height: 100px;
    box-sizing: border-box;
    margin-top: 10px;
    max-width: 100%; /* Makes it responsive */
}

.swiper-thumbnail .swiper-slide img {
    width: 100%;
    height: 100%;
    object-fit: cover;
    opacity: 0.5;
    cursor: pointer;
}

.swiper-thumbnail .swiper-slide-thumb-active img {
    opacity: 1;
}

3. JavaScript Logic

Initialize Swiper instances and handle the `mousemove` event for the zoom effect.

// 1. Initialize Thumbnails
const thumbSwiper = new Swiper('.swiper-thumbnail', {
    slidesPerView: 4,
    spaceBetween: 10,
    watchSlidesProgress: true,
});

// 2. Initialize Main Carousel linked to Thumbnails
const mainSwiper = new Swiper('.main-swiper', {
    loop: true,
    autoplay: {
        delay: 2000,
        disableOnInteraction: false,
    },
    navigation: {
        nextEl: '.swiper-button-next',
        prevEl: '.swiper-button-prev',
    },
    thumbs: {
        swiper: thumbSwiper,
    },
});

// 3. Pause Autoplay on Hover
const swiperContainer = document.querySelector('.main-swiper');
swiperContainer.addEventListener('mouseenter', () => mainSwiper.autoplay.stop());
swiperContainer.addEventListener('mouseleave', () => mainSwiper.autoplay.start());

// 4. Zoom Effect Logic
const targetDiv = document.getElementById('target-div');
const slides = document.querySelectorAll('.main-swiper .swiper-slide');

slides.forEach(slide => {
    // Show Zoom Div
    slide.addEventListener('mouseenter', () => {
        targetDiv.style.display = 'block';
        const img = slide.querySelector('img').src;
        targetDiv.style.backgroundImage = `url(${img})`;
    });

    // Hide Zoom Div
    slide.addEventListener('mouseleave', () => {
        targetDiv.style.display = 'none';
    });

    // Calculate Background Position
    slide.addEventListener('mousemove', (e) => {
        const rect = slide.getBoundingClientRect();
        const x = e.clientX - rect.left;
        const y = e.clientY - rect.top;
        const posX = (x / rect.width) * 100;
        const posY = (y / rect.height) * 100;
        targetDiv.style.backgroundPosition = `${posX}% ${posY}%`;
    });
});
